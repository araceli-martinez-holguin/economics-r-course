📌 Introducción

En esta práctica analizaremos datos climáticos históricos de distintos países utilizando la base de datos CRU TS 4.09 (Climatic Research Unit Time Series), desarrollada por la Universidad de East Anglia (Reino Unido). Esta base contiene registros mensuales desde 1901 hasta 2024 a escala de país.

Variables disponibles

Código Variable Unidad
pre Precipitación total mm
tmn Temperatura mínima media °C
tmp Temperatura media °C
tmx Temperatura máxima media °C

Objetivo

Al finalizar esta práctica serás capaz de:

  1. Descargar y leer datos climáticos en formato tabular con R.
  2. Calcular estadísticas descriptivas mensuales.
  3. Visualizar patrones climáticos con gráficos de barras.
  4. Reproducir el análisis para distintos países y variables.

📥 Descarga de datos CRU

Los archivos se descargan directamente del servidor de la Universidad de East Anglia.
El ciclo for revisa si cada archivo ya existe antes de descargarlo, evitando descargas repetidas.
Si un archivo no existe en el servidor, tryCatch captura el error y muestra un mensaje en lugar de detener la ejecución.

base_url <- "https://crudata.uea.ac.uk/cru/data/hrg/cru_ts_4.09/crucy.2503061057.v4.09/countries"

for (country in countries) {
  for (var in variables) {
    file <- paste0("crucy.v4.09.1901.2024.", country, ".", var, ".per")
    if (!file.exists(file)) {
      tryCatch(
        download.file(paste0(base_url, "/", var, "/", file), destfile = file, mode = "wb"),
        error = function(e) message("404: ", file)
      )
    }
  }
}

✅ Resultado esperado: Deberías ver los archivos .per en tu directorio de trabajo. Puedes verificarlo con list.files(pattern = ".per").

📂 Lectura de datos

Cada archivo .per tiene 4 líneas de encabezado que debemos saltar (skip = 4).
Los valores -999.0 indican datos faltantes y se reemplazan por NA para que R los maneje correctamente.
Todos los datos se almacenan en una lista llamada climate_data, donde cada elemento corresponde a un par país-variable.

col_names <- c("Year", "Jan", "Feb", "Mar", "Apr", "May", "Jun",
               "Jul", "Aug", "Sep", "Oct", "Nov", "Dec",
               "MAM", "JJA", "SON", "DJF", "ANN")

climate_data <- list()

for (country in countries) {
  for (var in variables) {
    file <- paste0("crucy.v4.09.1901.2024.", country, ".", var, ".per")
    if (file.exists(file)) {
      df <- read_table(file, skip = 4, col_names = col_names, na = "-999.0")
      df$Country <- country
      df$Variable <- var
      climate_data[[paste0(country, "_", var)]] <- df
    }
  }
}

🌧️ Precipitación mensual promedio — México

Cálculo de estadísticas descriptivas

Usamos sapply para aplicar una función a cada mes y calcular estadísticas sobre los 124 años de registro (1901–2024).

# Seleccionamos México y la variable precipitación
mexico_pre <- climate_data[["Mexico_pre"]]

# Vector de nombres de meses (coincide con las columnas del data frame)
months <- c("Jan", "Feb", "Mar", "Apr", "May", "Jun",
            "Jul", "Aug", "Sep", "Oct", "Nov", "Dec")

# Calculamos estadísticas por mes
mean_monthly <- sapply(months, function(m) mean(mexico_pre[[m]], na.rm = TRUE))
sd_monthly   <- sapply(months, function(m) sd(mexico_pre[[m]], na.rm = TRUE))
min_monthly  <- sapply(months, function(m) min(mexico_pre[[m]], na.rm = TRUE))
max_monthly  <- sapply(months, function(m) max(mexico_pre[[m]], na.rm = TRUE))

# Redondeamos a enteros para facilitar la lectura
mean_monthly <- as.integer(round(mean_monthly))
sd_monthly   <- as.integer(round(sd_monthly))
min_monthly  <- as.integer(round(min_monthly))
max_monthly  <- as.integer(round(max_monthly))

# Guardamos en una tabla
stats_mexico <- data.frame(
  Month = months,
  Mean  = mean_monthly,
  SD    = sd_monthly,
  Min   = min_monthly,
  Max   = max_monthly
)
print(stats_mexico)

✅ Resultado esperado: Una tabla con 12 filas (una por mes) y 5 columnas. Los meses de verano (Jun–Sep) deben mostrar los valores más altos de precipitación para México.

Gráfico de barras

png("bp_precipitacion_mexico.png", width = 800, height = 600, res = 100)

bp_mexico <- barplot(mean_monthly,
              names.arg = months,
              col       = "navy",
              main      = "Precipitación Mensual Promedio en México (1901-2024)",
              ylab      = "Precipitación (mm)",
              xlab      = "Mes",
              ylim      = c(0, max(mean_monthly) * 1.15),
              cex.names = 0.9,
              cex.axis  = 0.9,
              las       = 1,
              font.lab  = 2)

# Etiquetas con el valor sobre cada barra
text(x      = bp_mexico[, 1],
     y      = mean_monthly + 2,
     labels = mean_monthly,
     cex    = 0.8,
     col    = "black",
     adj    = c(0.5, 0))

# Línea de la media anual
abline(h   = mean(mean_monthly),
       col = "red",
       lwd = 2,
       lty = 2)

legend("topleft",
       legend = paste("Media anual:", round(mean(mean_monthly)), "mm"),
       col    = "red",
       lty    = 2,
       lwd    = 2,
       bty    = "n")

dev.off()
