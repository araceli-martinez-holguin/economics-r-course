## Course Overview
This repository contains R code, examples, and assignments for the Economics R Programming course.

## Folder Structure
- `laboratories/` - R scripts
- `data/` - Datasets used in laboratories/assignments
- `assignments/` - R scripts 

## Topics Covered
1. Data manipulation 
2. Economic data visualization 
3. Time series analysis 
4. Regression models 
5. Economic forecasting
6. Causal Inference

## How to Use
1. Clone this repository: `git clone https://github.com/araceli-martinez-holguin/economics-r-course.git`
2. Open in RStudio as a project
3. Run scripts in order
